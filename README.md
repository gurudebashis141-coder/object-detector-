# NeuralEye – AI Object Detection Website

NeuralEye is a real-time AI object detection web application that runs directly in the browser using TensorFlow.js.
It detects objects from the user's webcam and displays bounding boxes with confidence scores.

## Live Demo

https://debashisgurucamera.netlify.app

## Features

* Real-time webcam object detection
* Detects 80+ objects using the COCO-SSD model
* Displays bounding boxes and confidence percentages
* Works directly in the browser (no backend required)
* Mobile and desktop compatible
* Modern AI interface

## Technologies Used

* HTML
* CSS
* JavaScript
* TensorFlow.js
* COCO-SSD object detection model
* Netlify (deployment)

## How It Works

1. The website accesses the user's webcam.
2. TensorFlow.js loads the COCO-SSD model.
3. Each video frame is analyzed by the AI model.
4. Detected objects are displayed with labels and confidence scores.
